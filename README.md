# Mendixdemo

Responsive web application for app content management with role-based access.

## Run

```bash
npm start
```

Open `http://localhost:3000`

Demo users:
- Admin: `admin / admin123`
- User: `user / user123`

## Features
- Secure login with session cookie and role-based access control.
- Admin-only app upload/create/edit/delete and publish/unpublish controls.
- User read-only library for published apps.
- Dashboard, category browser, library filters/search/sorting, app detail view.
- Optional extras included: favorites, recently viewed, simple view analytics.
